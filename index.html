关于jQuery DataTables 的一些总结及参考资料

说真，这插件的确不大好用，提醒一下，能用其他就用其他吧。

  2. 创建行时可以添加样式： 
"createdRow": function(row,data,index){  
$(row).addClass("text-c")  
},
  1. 排序、搜索、翻页方法： 
$('.table').on('order.dt',function() {
    console.log('排序');
}).on('search.dt',function() {
    console.log('搜索');
}).on('page.dt',function() {
    console.log('翻页');
}).dataTable();
  1. 清空表格：
    $('.table').dataTable().fnClearTable();
  2. 增加数据：
    $('.table').dataTable().fnAddData(packagingdatatabledata(msgObj),true);
  3. 搜索某一行结果：
    table.columns(3).search("").draw();

关闭webpack的eslint验证

在使用vue-cli脚手架构建了webpack项目后，发现eslint的验证太tmd的严谨，实在不适合我这种懒癌症人员使用，所以必须要把它给关闭掉。

解决办法：

进入webpack.base.conf.js里，屏蔽以下代码：

preLoaders: [
  {
    test: /\.vue$/,
    loader: 'eslint',
    include: projectRoot,
    exclude: /node_modules/
  },
  {
    test: /\.js$/,
    loader: 'eslint',
    include: projectRoot,
    exclude: /node_modules/
  }
],

关于org.apache.jasper.servlet.JspServlet serviceJspFile 报错的解决办法

错误如下：

org.apache.jasper.servlet.JspServlet serviceJspFile 严重:PWC6117: File "null" not found
解决办法：

AdminController.java里所有返回路径最前面那个’/‘去掉．
比如 return "/admin/config/list";
修改为　return "admin/config/list";
